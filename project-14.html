Overview

A motion design guideline system built for Vervemobile, defining how the app's interface should move — from micro-interactions to full screen transitions — so animation stays consistent no matter who's building on top of it.

The guidelines set out core principles like timing, easing, and hierarchy of motion, then broke them down into reusable patterns for common UI moments: loading states, card transitions, button feedback, and navigation. The goal was a reference any designer or engineer could pick up and apply without needing to reinvent the feel each time.
