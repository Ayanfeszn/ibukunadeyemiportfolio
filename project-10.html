Africa API Product Launch

ClientAfrica API
RoleProduct Launch, 2D Motion Design
Year2026
ToolsAfter Effects, Illustrator

Overview

I directed the product launch video for Africa API — one REST API giving developers a single, consistent way to pull economic, market, and government data across all 54 African countries: indicators, FX and market data, trade flows, elections, policies, and official records.

My job was making a developer-first, data-heavy product feel immediate on launch day. I translated live signals, historical indicators, and one-consistent-contract API design into a short, confident motion piece that speaks directly to builders, with none of the technical detail getting in the way.
